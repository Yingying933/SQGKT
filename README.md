# SQGKT
The code is related to the paper SQGKT: Student-Question Interaction Graph-based Knowledge Tracing.

# Setup
The requiring environments is as bellow:
- Python 3.7+
- PyTorch 1.12.1
- Scikit-learn 1.1.2
- Numpy 1.21.5
- Pandas 1.2.4
